This is our help page!

Here you can find information that can help you navigate through our website.

In the Homepage you can see a list of coloured options. These are the different categories in which the DataSet which was analysed has been divided into. In order to view the data, select one of the options. This will lead you to another page in which you will either visualize the data directly or will be asked to pick a subcategory, which again will lead you to the page with displayed data.

In each page you will find a link to the Homepage making it easier to return to it and pick a different category of data.
